Создание референса.

Референс создавался в программе Krita — бесплатном и простом графическом редакторе. Примерный вид будущей страницы был набросан за 10 минут: определено расположение заголовка, блоков с текстом и картинок. Референс нужен только для понимания общей композиции и служил ориентиром при вёрстке.

Голый вид без CSS

После написания HTML без стилей страница выглядит неопрятно, все блоки прижаты к левому краю. На этом этапе проверяется правильность структуры: вложенность тегов и наличие всех элементов. «Голый вид» — это скелет будущей страницы, на который затем добавляется визуальное оформление.

Код страницы

Код страницы написан вручную и включает базовую структуру с тегами html, head и body. CSS подключается отдельным файлом, что делает код аккуратнее. Каждый блок обёрнут в div с понятными классами для удобства стилизации.

Страница без наполнения текстом и картинками

На данном этапе страница ещё не наполнена текстом и изображениями — присутствуют только пустые блоки с заданными стилями. Видна чистая архитектура страницы без отвлекающего контента. Блоки сохраняют своё расположение, фоны, отступы и рамки, но внутри них ничего нет.

Готовая страница

В финальной версии добавлены нормальные тексты и подходящие по смыслу изображения. Все блоки выровнены, цвета и шрифты подобраны для комфортного чтения. Разница между первым наброском и готовой страницей наглядно демонстрирует результат прохождения всех этапов.
