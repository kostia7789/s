О чём этот сайт

Данный сайт посвящён пошаговой инструкции по созданию сайта с нуля. Здесь наглядно демонстрируется каждый этап разработки: от первоначального референса до полностью готовой страницы. Сайт создан в рамках индивидуального проекта и показывает, как своими руками сделать рабочий веб-сайт без специального образования.

О проекте

Данный проект представляет собой пошаговую инструкцию по созданию сайта с нуля с практическим примером. В рамках работы разработаны несколько страниц: главная, страница с описанием документа, страница с демонстрацией всех этапов вёрстки и страница об авторе. Каждый этап создания сайта зафиксирован и описан, чтобы любой желающий мог повторить процесс самостоятельно.
